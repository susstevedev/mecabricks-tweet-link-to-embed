# mecabricks-tweet-link-to-embed
Replaces twitter.com links with embeds on Mecabricks
# How to install
Install [TamperMonkey](https://www.tampermonkey.net) or [Violentmonkey](https://violentmonkey.github.io).
Then click [here](https://github.com/susstevedev/mecabricks-tweet-link-to-embed/raw/main/index.user.js). You should see an installation prompt. Click install.
